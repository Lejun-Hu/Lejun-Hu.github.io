# 图片目录

请在此处添加你的图片资源：

1. **profile.jpg** - 你的个人照片（必须）
   - 建议尺寸：600x600 像素以上
   - 建议比例：1:1（正方形）
   - 格式：jpg 或 png

2. **project1.jpg**, **project2.jpg** - 项目截图（可选）
   - 建议尺寸：1200x800 像素
   - 建议比例：3:2
   - 用于项目展示区域

3. **favicon.png** - 网站图标（可选）
   - 尺寸：32x32 或 64x64 像素
   - 显示在浏览器标签页上

提示：如果你没有合适的图片，可以先使用占位图服务：
- https://via.placeholder.com/600x600?text=Your+Photo
- https://picsum.photos/600/600

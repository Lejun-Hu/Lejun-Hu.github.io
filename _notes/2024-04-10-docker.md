# Docker 常用命令速查

## 容器管理

```bash
# 运行容器
docker run -d -p 80:80 --name my-nginx nginx

# 查看运行中的容器
docker ps

# 查看所有容器（包括停止的）
docker ps -a

# 停止容器
docker stop my-nginx

# 删除容器
docker rm my-nginx
```

## 镜像管理

```bash
# 拉取镜像
docker pull nginx:latest

# 查看本地镜像
docker images

# 删除镜像
docker rmi nginx

# 构建镜像
docker build -t my-app:1.0 .
```

## 常用技巧

- 使用 `docker-compose` 管理多容器应用
- 定期清理无用镜像：`docker system prune`
- 查看容器日志：`docker logs -f my-container`
